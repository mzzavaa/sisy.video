# sisy.video

Speaker reels landing page for the SISY platform.

**Live site:** https://sisy.video

## About

Speaker reel hosting for women experts. Upload clips, build a searchable video portfolio, and let organizers see you before booking. Part of the [SISY ecosystem](https://sisy.world).

## Sections

- Hero - "Show them what you sound like"
- Speaker reel mockup cards (Highlights, Full Sessions, Podcast Clips)
- How It Works - Upload / Tag / Share (3-step)
- Features grid (Transcription, Search, Embeddable Widget, Analytics, Privacy)
- Pricing (Free: 1 video · Pro: €9/mo · Teams: custom)
- CTA

## Tech

Built with [Hugo](https://gohugo.io/). Deployed to GitHub Pages via Actions.

## Local development

```bash
hugo server -D
```

## Deployment

Auto-deploys on push to `main` via `.github/workflows/deploy.yml`.
